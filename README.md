EcoSort is a machine learning-based web application that classifies waste images into categories such as recyclable, compostable, and landfill. The goal is to promote smarter waste disposal and encourage sustainability through simple AI-powered feedback.

---

## 🌱 Project Motivation

Improper waste sorting is a major environmental issue. EcoSort was built to demonstrate how machine learning can be used in everyday tools to help users make more sustainable decisions.

---

## 🚀 Features

- Upload an image of waste material
- AI model classifies the item into a waste category
- Clean and simple web interface
- Instant prediction results
- Lightweight Flask backend for real-time inference

---

## 🧠 Machine Learning Model

- Trained using image classification techniques
- Preprocessed images for consistency and accuracy
- Model predicts waste category based on visual features
- Achieved strong performance on validation dataset

---

## 🛠️ Technologies Used

- Python
- Flask
- TensorFlow / PyTorch (depending on your model)
- HTML / CSS
- OpenCV (if used for preprocessing)
- NumPy / Pandas

---

## 💻 How It Works

1. User uploads an image through the web interface  
2. Image is sent to Flask backend  
3. Model preprocesses the image  
4. Prediction is generated in real-time  
5. Result is displayed on the webpage
